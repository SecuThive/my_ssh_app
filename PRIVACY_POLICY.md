# Pocket SSH 개인정보처리방침

**최종 수정일:** 2025년 12월 25일

**Pocket SSH**(이하 '본 앱')는 사용자의 개인정보 보호를 최우선으로 생각합니다.
본 문서는 본 앱이 사용자의 민감한 정보(서버 접속 정보, 키 파일 등)를 어떻게 처리하는지 명확하게 설명하기 위해 작성되었습니다.

## 1. 데이터 수집 및 전송 (No Data Collection)
**본 앱은 개발자의 서버로 어떠한 개인정보도 전송하지 않습니다.**
본 앱은 클라이언트 사이드에서만 동작하며, 사용자 데이터를 저장하거나 분석하기 위한 별도의 백엔드 서버를 운영하지 않습니다.

* **회원가입 없음:** 앱 사용을 위해 계정을 생성하거나 로그인할 필요가 없습니다.
* **추적 없음:** 사용자의 서버 접속 기록이나 앱 사용 패턴을 추적/수집하지 않습니다.

## 2. 데이터의 로컬 저장 (Local Storage)
앱 사용에 필요한 모든 데이터는 **사용자의 스마트폰 내부 저장소에만 저장**됩니다.

* **서버 접속 정보:** 사용자가 입력한 호스트(Host), 포트(Port), 사용자명(Username), 비밀번호(Password) 등은 기기 내부에 암호화되어 저장됩니다.
* **SSH 키 파일:** 사용자가 불러온 Private Key (PEM, PPK 등) 파일은 앱 내부의 격리된 공간(Sandbox)에 안전하게 보관됩니다.
* **사용 목적:** 위 정보들은 오직 사용자가 지정한 서버에 SSH 접속을 수행하기 위한 용도로만 사용됩니다.

## 3. 앱 권한 안내 (App Permissions)
본 앱은 기능을 수행하기 위해 최소한의 권한만을 요청합니다.

* **인터넷 (INTERNET):** 사용자가 지정한 외부 서버와 SSH 연결을 맺기 위해 필요합니다.
* **저장소 읽기 (READ_EXTERNAL_STORAGE):** (선택 사항) 사용자가 기기에 저장된 키 파일(Key File)을 앱으로 불러올 때만 요청됩니다.

## 4. 오픈소스 투명성 (Open Source Transparency)
본 프로젝트는 **오픈소스(Open Source)**로 공개되어 있습니다.
사용자의 데이터가 외부로 유출되지 않음을 소스 코드를 통해 직접 검증하실 수 있습니다. 우리는 '보안'과 '투명성'을 가장 중요한 가치로 둡니다.

## 5. 제3자 서비스 (Third-Party Services)
본 앱은 배포 및 안정성 확보를 위해 **Google Play Services**를 이용합니다. 이에 따라 구글의 개인정보처리방침이 적용될 수 있습니다.
